Lin Project Radar — calibration verification

Re-verification script for calibration/manifest.json. Loads the manifest (authored target_status per synthetic demo project) and the LIVE project list from the deployed backend (?action=listslim, via window.LIN_API_URL) and compares the computed (live) status against each project's manifest target. PASS means the live computed status matches the manifest's target_status. A manifest entry with target_status: "unset" is NOT treated as a pass — it renders a distinct "NO TARGET SET" badge, since no claim about correctness can be made until a human confirms the intended target from the authored documents. Run this after any module/signal-math change to confirm calibrated projects still land where they were designed to land.
