Handbook → chatbot _lib export

Single-source export tool. Reads the Handbook content source (assets/js/knowledge.js, the same content the in-app Handbook renders from) and deterministically transforms it into the five plain-text files the chatbot's Drive _lib folder is meant to hold. Output is GENERATED — re-run this page after any Handbook content change and manually re-upload the five files below (Drive is not reachable from this repo/environment, so uploading stays a manual step). Nothing here calls a network endpoint; everything runs against the already-loaded knowledge.js content in your browser.
